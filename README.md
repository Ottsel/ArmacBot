# ArmacBot
Golang Moderator and Soundboard Bot for Discord

# Prerequisites

* gcc
* ffmpeg
* opus

# How to setup

Install the bot and run once to generate a config file, then Configure at least ```BotToken``` and ```GuildID```
```
go get github.com/ottsel/armacbot/
```
